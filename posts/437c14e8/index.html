学英语的经验教训总结

发表于 2020-07-12 | 更新于 2020-07-13 | 分类于 经验教训 | 评论数：

一个学英语没天赋的娃硬硬地把英语磨下来的血泪史。

我之前备考的是托福，那就是听说读写，外加一个背单词。挨个说。

背单词

我是2019.7.18开始坚持背单词的，用的软件是百词斩，从六级单词开始，到托福词汇，从一天背60个到一天背200个，一直坚持到12月，期间从未停过。

因为单词记不住，我已经数不清有多少次深夜里一个人伏案痛哭，要知道我之前从来没有因为学不会而哭过，包括高考，包括考研，我都是从容且自信。

我之前用的是百词斩，每次都是完成拼写，看图识词，听音辨词，看词选意这四项，结果收效甚微！

后来改用【不背单词】，通过不断地反复，不断地重复，虽然同样时间完成的单词数更少，但是质量更高，通过不背单词完成的单词，我能在一两个月以后还记得意思，而且它的复习更科学，更符合记忆曲线，推荐使用！

另外我看到的一个B站Up主上传的A4纸背单词法，其原理也是通过不断地重复，来记住单词，我没试过，但是我觉得效果应该也不错。

阅读

阅读的基石是词汇，所以单词不行阅读抓瞎。

技巧方面，我是看的智课网的【岳建辉托福阅读解题技巧精讲班】，中科大学生免费。看完以后，再去做托福的阅读，感觉正确率提高了很多。建议在熟悉了托福阅读的基本题型后，就看一遍这个课（或者其他类似的课）。

以下是我当时听课记的一些笔记摘要：

  1. 文章：
    1. 主题：一篇阅读有且只有一个主题，一般出现在标题上。
    2. 态度，一般有三种，positive，negative，neutral。默认是neutral。
  2. 段落：
    1. 段落由句子组成，段落中的句子分两类：exemplification 或者explanation，就是主题句或者解释主题句的句子。
    2. 段落类型：
      • 金字塔型，主题句在开始部分
      • 漏斗型，主题句在后面
      • 铅笔型，前面过程基本上是一直，最后得出一个结论，观点和论证在同一个句子中。
    3. 段落关系
      • 并列：同一话题，不同方面。
      • 顺承：同一话题，同一方面。
      • 类似于语文中，给文章分段。
  3. 句子：
    1. 句子分类：简单句和非简单句。简单句就是只有一套主谓结构的句子。非简单句就是有两套，甚至更多的主谓结构。
    2. 句子中，心脏是谓语动词，标志着一个句子。
    3. 连接词：能把两个句子连接在一起的词，都叫连接词。
    4. 句子之间的关系：并列和嵌套。嵌套：句子之间有修饰关系。其余的都是并列关系。
    5. 如果是嵌套关系，越在外面的句子越重要，是主句。
  4. 词汇：
    1. 做题时，遇到不会的单词，不要立马查词典，尽量猜测词语的意思，最后再查词典。

旧版的托福iBT考试，每篇阅读一般是13个题，新版的就只有10个题了，这套课程对应的还是老托福，不过新旧托福只是题目数量变了，题型不变，做题方法依然适用。

  1. 词汇题
    1. 题目类型，识别单词的意思，The word X in the passage is closest in meaning to… ，In stating X, the author means that…
    2. 基础：没辙，老老实实背单词。
    3. 技巧/方法：
      1. 介词搭配，找有没有固定的介词搭配
      2. 文本逻辑：近义词：并列，解释，举例。反义词：转折。
      3. 提取主干：名词/形容词/动词/副词
  2. 句子题
    1. 常见的是化简句子。
    2. 常见的选项错误原因：改变意思，丢掉部分关键信息。
    3. 做题技巧：把句子拆开，把一个大的，非简单句拆开，拆开成若干个简单句，划出框结构去答题。
    4. 要点1：圈出主要逻辑关系，逻辑是句子中不可变的东西，会在正确答案中重现。不可改写的逻辑：肯定（注意双重否定），否定。可改写的逻辑：转折，因果，比较，对比。
    5. 要点2：排除错误选项，错误选项一般是：假，大，错。假：虚假的逻辑关系，虚假的修饰。大：将细节放大。错：跟原文矛盾，或者引入了原文根本没有的信息。
  3. 指代题
    1. 代词一定指代名词
    2. 单复数一致原则
    3. 前指原则，指代的都是前面出现的单词。
    4. 最基本的：讲得通，make sense原则。
  4. 文本插入题
    1. 句子分开的时候，要完整的分开。即两个句子要两边相对完整。被插入句子提出的概念，要与前面/后面句子有相类似的概念。
    2. 通用解题法：
      1. 拆分，简单句和非简单句，拆分方法是不一样的。简单句的拆分方法，通常从谓语动词的地方拆开。非简单句从关联词之间拆开。保证两边概念相对完整
      2. 提取，找到一个相对完整的概念，两个原则：被拆分句子中存在代词时，这个代词会往前指，这个是最常见的线索，第二个原则：逻辑关联概念。提取概念的时候，概念在上文中被重复的概率越高，越应该被优先选择。被拆分句子中找到一个概念，在上文中和下文中都重复出现。一定要找到事实依据。
      3. 对应，在上下句中找到与之相匹配的概念。找到概念的等于，约等于，属于关系判断结合条件能不能成立
      4. 排除，两种可能性：提取出对应的概念之后，能很顺利找到上下文中匹配的概念。或者当前后句子之间有明确的指代关系，这时候就是逻辑严密，没法插入句子。
  5. 信息题
    1. 包括事实信息题，否定事实信息题，推断题，修辞目的题。
    2. 首先要区分事实和观点。
      1. 事实：包含的内容基于某种直接证据，比如官方文件，参考书，法律记录，经验或观察。通常，事实可以提供有关人物，地点和事件的信息，但不会传达作者个人观点或感受。
      2. 观点：观点的陈述通常表达个人信念、感情、态度、价值观、看法，或一个人对某个主题或者话题所作的判断，通常，观点表现为推论，预感，猜测或结论等形式。
      3. 如果一个表达式观点，它的反面也可能同时成立。
      4. 观点还有一个特别明显的标志：带有主观色彩的形容词
      5. 我们在托福阅读和题目的选择当中，选择的都应该是事实，而不是观点
    3. 做出有效推断
      1. 有效推论是指基于清楚地证据和符合逻辑的推理而得出的结论或概括
      2. 无效推论是指证据不妥或基于错误逻辑而进行的假设。
      3. 无效推断的两个错误原因，1， 证据不妥。原文中没有明确的事实支撑。 2， 逻辑错误，原文中没有这样的逻辑，原文中的比较关系被颠倒了。
      4. 学会侦探式思考：reading between the lines。在字里行间寻找线索。
    4. 常见的干扰项设置原理
      1. 引入错误信息（位置 内容）
      2. 建立虚假逻辑：
        • 乾坤挪移 ，位置的错误，问的第三自然段，引入了第二自然段的内容
        • 无中生有：引入了原文未出现的信息
      3. 猪逻辑：错误的逻辑，原文中未出现的逻辑，原文中出现的逻辑找到了修改
    5. 正确选项设置原理
      1. 原文重现
      2. 同义改写
      3. 有效推论
    6. 事实信息题与否定事实信息题解题技巧：关键词对应。这个题基本上不会融合任何个人观点的。
    7. 推断题考试技巧
      1. 推断题考察推理过程，重点考察排中律，非重点考察溯因，归纳，演绎。
      2. 排中律：排除中间的定律，非此即彼，非黑即白，没有中间选项。排中律适用于：同一集合中，A和非A，占据了全部概念。这样的一种状况。所以如果概念A出现在文中的话，我们就可以用排中律。另一种情况：原文中出现了表示时间点的概念。
      3. 绝大多数的推断题，都可以当做细节题来理解，所以都要重视同义改写。
      4. 排中律：使用逻辑关系的话，要使用排中律。A以及非A，带时间点的
      5. 最后采用一般考点，常考的是归纳。
    8. 修辞目的题，当做结构分析+细节题来思考，题目相对简单，一般涉及到的就是文章的结构。
  6. 六选三或者图表题
    1. 理清段落关系，每个段落中的keyword 记录下来，然后关联成一个线路图。根据段落关系来解题。
    2. 利用介绍句来排除部分选项。

听力

使用考满分或者小站托福，反复的听。然后我也听了智课网的网课，感觉作用不是特别大。

写作

依然用了科大学子免费的智课网的作文批改服务，然后多背范文。

多邻国

2020年对于留学生太惨了，托福考试从2月一直取消到6月，即使7月也只有极个别的考场开放。

我要申请的学校于3月下旬开始说接受多邻国的成绩，我有火急火燎转去备考多邻国，一开始连多邻国是啥都搞不清楚。具体介绍不再赘述，这里说一下我备考多邻国时的经验教训。

  1. 知乎上有个老师叫【米斯特潘】，不是我吹，他的一些经验技巧确实有用，包括出分技巧，题型重点等，建议好好看看他的专栏。
  2. 单词与托福单词有很大不同，托福单词更多的是学科词汇，但是多邻国的词汇是按照欧标的，叫CEFR词汇标准。但仅仅靠这个词单是不够的，我又找了多家按照CEFR分类的词汇表，提取了B2~C2的词单，赠与有缘人，见文末。
  3. 真题练习建议花钱买【登登多邻国】的真题，和考试的形式很像，而且里面的看图说词环节，覆盖了题库的绝大部分，建议背熟这一部分，就是来送分的。
  4. 完形填空绝大部分，在网上都有原文，使用谷歌搜索，用双引号精确匹配，很容易搜到。
  5. 口语部分，我是参加的【米斯特潘】组织的免费的英语角，我感觉对我提升挺大，从一开始不敢开口到敢于表达自己的观点了。
